# Soundbyte Web App

First, run the development server:

```bash
npm i
npm run dev
```

### Frontend Tech Stack

This project uses the BunJs runtime environment.

- Components & Routing: NextJs/React
- UI Library: Shadcn
- CSS: Tailwind
- Data Type Validation: Zod
- ORM: Drizzle
- OAuth: Better Auth (session)

### Development

Database Migrations

```sh
npm run db:generate
npm run db:migrate
```
